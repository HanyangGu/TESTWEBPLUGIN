Download this folder
Open Google Chrome 
Click on three dots on the top right, then look for "Manage Extensions"
Enable "Developer Mode" on the top right
Click on "Load Unpacked" on the top left
Select "ai-pdf-uploader" folder
Open Canvas and Log in
Click on extension tab on the top right and click on "AI PDF Uploader"
Click on config tab on the top right of the "AI PDF Uploader" pop up 
enter OpenAI API and Save
Now upload a pdf file contains only text to the "AI PDF "Uploader"
Enter "Instruction for AI", Such as "Navigato to NLP class, find recommendation engine assignment and submit"
Click "Start AI Agent"
